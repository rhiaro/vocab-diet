Abstract

This vocabulary describes common dietary constraints, and terms to express them with regards to a person. There are other vocabularies for linking recipes, dishes or restaurants to suitability for particular diets.

Discussion

Notes about things considered when creating this vocabulary.

  • Diets are of type schema:RestrictedDiet even though this is an enumerated list of a fixed set of values, because that constraint is dumb.
  • No domain or range on the properties, because whilst they're intended for some kind of person, I'm not interested in dicating what RDF type that person is. Simiarly I'd like people to publish their own dietary constraints at URIs and use this as objects where necessary. Also I'm happy for the object to be a literal where someone is too lazy to publish a constraint with a URI (eg. very specific/niche things like "No cooked tomatoes"; really I know someone allergic to cooked tomatoes but not raw ones).
  • Some diets make sense in both 'no' and 'low' form, and some only really make sense as one or the other (to me) but they all have both options, because who am I to decide which make sense. Except 'calorie'. There's no such thing as a no-calorie diet, right?
  • Definitions of 'low' vary between regions/cultures/regulatory bodies. Use context to determine..
  • I considered creating classes for 'requirement', 'strong preference' etc, and reifying dietary constraints, but decided creating properties was ultimately less work for me right now, and for publishing this data. If one day I need to add extra properties to a dietary constraint (ie. 'I was vegan between these dates') then I might rethink this model, but I'm not going to overengineer in advance.

If I've missed something PR/issue welcome on github.

Properties

Dietary constraint

A constraint (restriciton or preference) of a person for a particular diet.

Critical dietary requirement

A requirement of a person for a particular diet, with no risk of cross-contamination (possibly for medical reasons).

subProperty of: dietary constraint

Dietary requirement

A requirement of a person for a particular diet, with no likelihood of compromise.

subProperty of: dietary constraint

Strong dietary preference

A strong preference of a person for a particular diet; compromise or blissful ignorance may be permissable in some circumstances.

subProperty of: dietary constraint

Weak dietary preference

Some preference of a person for a particular diet, but can be flexible or probably won't go out of their way to achieve this.

subProperty of: dietary constraint

Instances

None

No dietary constraints

Pescetarian

Does not eat meat, but eats fish/seafood.

Lacto-vegetarian

Vegetarian, eats dairy but no eggs.

Ovo-vegetarian

Vegetarian, eats eggs but no dairy.

See also: schema:VegetarianDiet, dbpedia:Vegetarianism

Vegan

Vegan; no animal products including eggs, dairy and honey.

See also: schema:VeganDiet, dbpedia:Veganism

Halal

A diet conforming to Islamic dietary practices.

See also: schema:HalalDiet, dbpedia:Halal

Kosher

A diet conforming to Jewish dietary practices.

See also: schema:KosherDiet, dbpedia:Kashrut

Fruitarian

A diet that includes fruits, vegetables, nuts, and seeds, without animal products or grains.

See also: dbpedia:Fruitarianism

Raw

Uncooked, unprocessed or simply processed foods.

See also: dbpedia:Raw_foodism

Macrobiotic

A dietary regimen which involves eating grains as a staple food, supplemented with other foods such as local vegetables, and avoiding the use of highly processed or refined foods and most animal products.

See also: dbpedia:Macrobiotic_diet

Organic

Foods produced by organic farming. While the standards differ worldwide synthetic pesticides and chemical fertilizers are not allowed.

See also: dbpedia:Organic_food

Usage

This vocabulary is used by (and was initially developed for) Foodulator, for storing your dietary constraints so that other apps can read them from your profile. Example:

@prefix diet: <https://apps.rhiaro.co.uk/foodulator/terms#>
<http://rhiaro.co.uk/about#me>
                               diet:dietaryConstraintCritical diet:Ovovegetarian, diet:Lactovegetarian ;
                               diet:dietaryConstraintRequired diet:Vegan ;
                               diet:dietaryConstraintRequired diet:AlcoholNo ;
                               diet:dietaryConstraintStrong "No walnuts" ;
                               diet:dietaryConstraintStrong "No raw onion" ;
                               diet:dietaryConstraintStrong "No pineapple" ;
                               diet:dietaryConstraintWeak diet:SugarLow ;
                               diet:dietaryConstraintWeak diet:Raw ;
                               diet:dietaryConstraintWeak diet:SaltLow ;
                               diet:dietaryConstraintWeak diet:Organic .

If you start using it for something or publishing your dietary constraints somewhere using this, let me know or PR to add yours here.
